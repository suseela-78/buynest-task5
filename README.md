# 🛒 BuyNest – Online Shopping Website

## 📌 Project Overview

BuyNest is a full-stack e-commerce web application developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP. The project allows users to register, log in, browse products, add items to the cart, and place orders through a simple checkout process. It also includes an admin panel to manage products and users.

## 🚀 Features

* User Registration & Login
* Product Listing
* Product Details Page
* Add to Cart
* Shopping Cart Management
* Checkout System
* Order Placement
* Admin Dashboard
* Product Management (Add, Edit, Delete)
* User Management
* Responsive UI Design

## ▶️ Installation

1. Clone the repository.
2. Copy the project to the XAMPP `htdocs` folder.
3. Import the database into phpMyAdmin.
4. Update database credentials in `db.php`.
5. Start Apache and MySQL.
6. Open `http://localhost/BuyNest`.
